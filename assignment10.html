Answers to Questions

  • I used a free Cat Fact API. It provides users with information about different cat breeds, as well as random cat facts.
  • I found it on https://catfact.ninja/.
  • There were two different pieces of information you could pull from this site, cat breeds and cat facts. I decided to pull cat breeds and found data on the names, countries, and patterns of different breeds. I had to use a limit key which specified how many cat breeds I would like to access.
  • This API could be useful if you own a cat cafe, where people can get coffee and pet cats. You could display random cat facts on your website to entice people to come visit your shop. This API could also be used by a cat adoption agency. The organization could use it to display information about different cat breeds in order to inform adopters about their cat selections.
